# Sheldon Rock Paper Scissors (Big Bang Theory Edition)

A fun Python command-line game based on the extended version of Rock–Paper–Scissors as seen in *The Big Bang Theory*, featuring:
- Rock
- Paper
- Scissors
- Lizard
- Spock
